# galeria-de-jogos
Este projeto é uma vitrine interativa de jogos, criada para demonstrar habilidades em desenvolvimento front-end. A aplicação permite que os usuários explorem um vasto catálogo de jogos, assistam a trailers, favoritem títulos e muito mais. É uma plataforma totalmente responsiva, construída do zero para oferecer uma experiência de usuário fluida e visualmente atraente.
